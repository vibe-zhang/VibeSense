# VibeSense 用户手册

## 1. 产品概述
VibeSense 是一款低成本、高精度的设备振动监测节点，适用于电机、风扇、水泵等旋转机械的早期故障预警。

## 2. 技术规格
| 项目 | 参数 |
|:---|:---|
| 采样率 | 1000 Hz |
| FFT 点数 | 256 |
| 频率分辨率 | ~3.9 Hz |
| 检测故障类型 | 不平衡 |
| 通信方式 | Wi-Fi (MQTT) / 串口 |
| 供电 | 5V Micro USB |
| 功耗 | < 0.5W |

## 3. 安装步骤
1. 将传感器用双面胶或磁吸底座固定在设备轴承座或外壳上（Z轴垂直于被测表面）。
2. 连接电源。
3. 设备自动连接预设 Wi-Fi，开始监测。

## 4. 指示灯说明
| LED 状态 | 含义 |
|:---|:---|
| 绿灯常亮 | 正常运行 |
| 红灯闪烁 | 检测到故障 |
| 红绿交替 | Wi-Fi 连接中 |

## 5. 故障判断逻辑
设备每 5 秒采集一组振动数据，提取 11 个时域/频域特征，通过内置的随机森林模型进行推理。当投票结果显示为故障时，立即触发报警。

## 6. 数据格式（串口输出）
✅ 正常 | 主频:50.2Hz 幅值:0.45
⚠️ 报警 | 主频:50.3Hz 幅值:1.82 (超出阈值)

## 7. 常见问题
**Q: 传感器数据不动？**  
A: 检查 I2C 接线，确保 VCC 为 3.3V。

**Q: 频繁误报？**  
A: 重新采集正常基线数据，调整 `scaler_mean` 和 `scaler_scale`。
